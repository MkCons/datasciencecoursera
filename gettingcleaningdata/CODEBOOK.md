# Data Dictionary

Activity

    Name of the activity performed by the subject. Range of values:
    - LAYING
    - SITTING
    - STANDING
    - WALKING
    - WALKING_DOWNSTAIRS
    - WALKING_UPSTAIRS
    
Subject.Identifier

    Code that uniquely identifies the subject. Range of values:
      01...30
